# Alan Rebek — Entrenador de Futsal

Sitio web personal de Alan Rebek, entrenador de futsal de primera división en Argentina.

## Contenido

- Presentación y valores de juego
- Trayectoria como jugador y entrenador
- Palmarés y títulos
- Galería de momentos icónicos
- Tablero táctico interactivo (canvas HTML5)
- Planes de mentoría integral 2026
- Referentes metodológicos
- Sección de contacto

## Tecnología

- HTML5 / CSS3 / JavaScript vanilla
- Un solo archivo `index.html` (self-contained, sin dependencias externas salvo Google Fonts)
- Canvas interactivo para tablero táctico con exportación a PNG

## Cómo usar

1. Abrí `index.html` en cualquier navegador
2. Para publicar en Netlify: arrastrá `index.html` en [netlify.com/drop](https://netlify.com/drop)
3. Para GitHub Pages: activá Pages en Settings → Pages → rama `main` → `/root`
